# Todo CLI

A simple command-line ToDo list manager using Python.

## Usage

Add a task:
```sh
python todo.py add "Buy groceries"
```

List tasks:
```sh
python todo.py list
```

Remove a task:
```sh
python todo.py remove 1
```

## Requirements
Python 3.x
